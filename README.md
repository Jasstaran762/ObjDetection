# Object Detection using YOLO

A real-time object detection project built using YOLO and Python.

## Features

- Real-time object detection
- Webcam/video support
- YOLO-based detection
- Fast inference
- Bounding box visualization

---

## Tech Stack

- Python
- OpenCV
- YOLO
- Ultralytics

---

## Installation

Clone the repository:

```bash
git clone git@github.com:Jasstaran762/ObjDetection.git
cd ObjDetection
```

Create virtual environment:

```bash
python -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run the Project

```bash
python main.py
```
